#서울시_관광지_정보_제공 #서울나들이

프로젝트 설명

"서울나들이"는 서울시 관광지 데이터를 사용하여 여러 정보를 제공해주는 사이트입니다.
또한, 관광지에 대한 커뮤니티가 존재하며, 작성된 게시글은 조회수와 별점을 통해 평가가 가능합니다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
- 적용 공공데이터(OpenAPI) : 서울시 관광지 정보
- 기타 : Kakaomap API, Bootstrap
